# Solana-Trader-Scanner
Solana Smart Wallet Finder is a Python-based tool that extracts and analyzes high-activity Solana wallet addresses using data sourced from the Nansen API.

The script is designed to collect large volumes of wallet addresses (100k+), focusing on identifying active participants in on-chain activity such as swaps and trading behavior. It enables further analysis of wallet performance, activity levels, and transaction patterns.

This filters:
Transactions on Solana
For a specific token
Within a date range (~30 days)
Only trades between $100 – $100k
